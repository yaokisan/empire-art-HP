# 素材格納フォルダ構成

## フォルダ構造
```
assets/
├── images/     # 画像素材
├── fonts/      # フォントファイル
└── videos/     # 動画素材

content/
├── text/       # テキストコンテンツ
└── data/       # データファイル
```

## 必要な素材チェックリスト

### 画像素材 (assets/images/)
- [ ] Empire Artロゴ（SVG形式推奨）
- [ ] Empire Artロゴ（PNG形式 - 透過背景）
- [ ] 代表写真または理念を表現する抽象画像
- [ ] BEAUTY ROADのキービジュアル
- [ ] オフィス風景または働く様子（任意）
- [ ] ヒーローセクション用背景画像

### テキスト素材 (content/text/)
- [ ] 代表メッセージ（100-200字）

- [ ] Media Creation事業説明（200字程度）
- [ ] AI Products事業説明（200字程度）
- [ ] 採用情報（任意）
